# prep-javascript-conditionals-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What does `strictly equal` mean?
  Strictly equal means the two operands are also of the same type. I.e 10 == "10" is true, but 10 === "10" is false.
- What is the logical **and** operator?
  &&, used for testing if multiple boolean operands are true
- Can you name some comparison operators?
  Yes I can. (jk, some are equal to ==, less than <, greater than >, less than or equal to <=, not equal to !=, etc.)
